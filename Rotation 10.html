Raspberry Pi

Raspberry Pi consists of physical computing. This means coding hardware systems such as computers, arduino, and in this unit, Raspberry Pi. This is a model of the Raspberry Pi Boards.

Coding

In class, we were able to code our own Rasperry Pi. The boards were like mini computer software program that we hooked up to a larger computer. For example, one project we did was code so that by the push of a button you could take a picture with a tiny camera that projected onto the bigger computer, take bursts, and change the edit of the pictures.
